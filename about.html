See GoCode in action:

The first online IDE that works offline:

 As simple and fast as installing a Chrome extension, working also when connection is poor.

Gocode is everything you need to start coding in 3 independent windows:

 

 

1- File browser

 

A simple but complete file explorer:

- Drag and drop features

- Sync with local folder

(can be Dropbox or any other cloud storage)

- Full GIT support

- Create and save new files

 

 

 

 

2- Text Editor

 

 Sublime-like text editor:

-Fully compatible with most

coding languages

-Autocomplete function

-Tabs/windows

-VIM/EMACS shortcuts

-Themes

 

 

 

 

 

3- Commander

 

- The center of control of GoCode

 in a Terminal style window

- Run a static server, Connect to our

 servers, GitHub and much more
